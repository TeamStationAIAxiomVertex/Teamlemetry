## Implementation Rule

V1 must prioritize telemetry fidelity first:

1. GitHub, Jira, and Slack metadata ingestion
2. normalized telemetry events
3. topology graph engine
4. probability and queueing engines
5. delivery confidence
6. evidence-backed recommendations
7. Slack reporting and MVP dashboard

No employee surveillance. No keystroke tracking. No activity policing. No employee-worth scoring.
